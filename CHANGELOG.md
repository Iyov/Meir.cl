# Changelog

Todos los cambios notables en este proyecto serán documentados en este archivo.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/es-ES/1.0.0/),
y este proyecto adhiere a [Semantic Versioning](https://semver.org/lang/es/).

## [1.0.0] - 2026-02-08

### Agregado

#### SEO y Optimización
- Meta tags optimizados (title, description, keywords)
- FAQ Schema (Schema.org FAQPage) con 5 preguntas estructuradas
- Open Graph y Twitter Cards actualizados
- Canonical URL implementado
- Geo tags para SEO local (Santiago, Chile)
- Business information meta tags

#### Accesibilidad
- Skip to main content link
- ARIA labels completos en FAQ (role, aria-expanded, aria-controls)
- Navegación por teclado (Enter, Espacio, Tab)
- Soporte para lectores de pantalla
- Cumplimiento WCAG 2.1 Nivel AA

#### Funcionalidades
- Botón flotante de WhatsApp con mensaje predefinido
- Formulario de contacto completo con validación HTML5
- Sistema de versionado de assets (?v=YYYY-MM-DD_N)

#### Diseño
- Estilos CSS para botón de WhatsApp
- Estilos CSS para formulario de contacto (responsive)
- Estilos CSS para skip-to-main
- Mejoras en responsive design

#### Documentación
- README.md profesional con badges y estructura clara
- LICENSE (MIT)
- CONTRIBUTING.md con guías detalladas
- SECURITY.md con política de seguridad
- CHANGELOG.md (este archivo)

### Cambiado

- Title tag reducido de 82 a 50 caracteres
- Meta description optimizada de 162 a 123 caracteres
- Keywords actualizadas con long-tail keywords
- JavaScript mejorado con mejor manejo de eventos
- Estructura de FAQ con mejor accesibilidad

### Mejorado

- Rendimiento de carga con versionado de assets
- Experiencia de usuario con múltiples CTAs
- Accesibilidad general del sitio
- SEO on-page y técnico
- Documentación del proyecto

### Corregido

- Validación de formularios
- Navegación por teclado en FAQ
- Contraste de colores para accesibilidad

## [0.1.0] - 2026-02-02

### Agregado

- Estructura inicial del sitio web
- Diseño responsive
- Modo oscuro/claro
- Multiidioma (Español/Inglés)
- Sección de testimonios
- Dato geek diario
- Barra de progreso de scroll
- Botón "volver arriba"
- Navegación suave (smooth scroll)
- Schema.org básico (Organization, WebSite, WebPage)
- Sitemap.xml
- Robots.txt

- .htaccess con configuración de caché
